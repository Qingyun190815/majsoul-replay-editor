# 龙之目玉模式

活动场对应 `mode_id` 是 35, 关键词是 `muyu_mode`

## 规则说明

- 对局开始时随机有1名玩家获得1枚"目玉"
- 某玩家拥有"目玉"时, "目玉"会计数, 该玩家每切牌1次, 计数加1
- 计数达到5时, "目玉"会随机转移到另一个玩家, 原玩家不拥有, 转移后的玩家拥有(注意玩家可能相同)
- 拥有目玉的玩家点数结算翻倍(不包括流局部分, 包括和牌, 放铳, 被自摸)
- 各家起始点数 40000

## 函数特殊说明

### 目玉位置 `muyuseats` 

指定一小局目玉出现的位置顺序, 若顺序队列已用完, 则目玉位置随机刷新

例子: `muyuseats = "1230"`
