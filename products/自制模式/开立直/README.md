# 开立直

开立直即"开牌立直"或"open立直", 在立直的同时展开手牌对其他玩家可见, 村规, 在某些影视作品出现过

开立直与一般的立直相比, 有以下区别:

- 若放铳给开立直家的玩家如果没有处于立直状态, 则看成"故意放铳", "开立直"记"役满"
- 其他开立直家和牌的情形(放铳家已经立直, 或自摸), 与一般的立直相比, 多记1番

使用开立直需要在 `qiepai` 函数中注明参数 `"kailiqi"`, 如 `qiepai("kailiqi")` (不需要再加 `true` 表示立直)
